Comparison Between Different Compression Techniques using Various Encoding Methods (12/2017 – 04/2018)

  • The paper discussed the different encoding techniques used in computer communication, and aimed to draw comparison between them, by comparing how much the techniques compressed images and text before transmission. The different techniques used to draw out the comparison were: Run Length Encoding (RLE), Huffman Encoding, Lempel-Ziv Encoding and Discrete Cosine Transform Encoding (DCT). Through the simulations, the various outputs are compared for their compression ratio, space savings and quality loss – and through the comparison, we tried to arrive at a conclusion as to what effect the different encoding techniques had on the input data stream.
  • In image encoding, it was seen that for large sized images, DCT was the best option for encoding (and compression). However, below a certain file size, RLE was still the better method. Huffman encoding was seen to be slightly of the “in-the-middle” type technique, however it has the most variations and can be easily re-coded to achieve an even better compressibility ratio.

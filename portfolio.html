Cabin in the Woods

Unreal Engine 4

A level design project, that primarily focuses on environment design. Primary goals were to work with foliage (LoD levels) and terrain design tools, as well as lighting scenarions. It also includes a log cabin, as well as smoke and fire particle effects. The First Person Shooter template of Unreal Engine has been used here, to allow the player to roam about, and experience the environment.

Master Bedroom

Trimble SketchUp | Unreal Engine 4

A digital recreation of a bedroom of a house. The initial design was done in Trimble SketchUp, and then the projet was imported into Unreal Studio using the new “.udatasmith” extension. The primary focus here was to understand the tools and working of SketchUP, and the workflow between a 3D modelling software and a game engine.

Escape the Room

Unreal Engine 4

An escape room scenario based level. Primary focus was on building a physics based system inside Unreal. A grabbing system was implemented, which would allow users to pick up and place different objects (each have a specific "Weight" value atttached to them). The exit opens if a set amount of weight is placed in a particular area (a trigger-box).

Jungle Run

Unreal Engine 4 | Adobe Photoshop

A game level that is modelled after a 2D 16-bit sidescroller, from back in the day. Primary focus was to implement classic gameplay mechanics, as well as employing fundamental concepts of game design such as visual waypathing, lane design, and progress structure. A minor goal for me was to figure out the nuances of game design, where I tested out the risk vs. reward concept that has been prevalent in video games for such a long time. The game has been specifically designed in such a way, that it provides players with a rewarding scenario, provided they are willing to take a risk to get to the zone.

Lost in Space

Unity3D

This virtual reality demo was created as a part of the edX course I did, entitled “Creating Virtual Reality Applications” (UCSanDiegoX:CSE190x). A major focus was to build a "one-button" system for virtual reality based environments. The application comprises of two parts - creation of game objects in a virtual room and manipulating them (along with existing objects in the scene). The user is able to freely create objects and them relocate, rotate, and scale them, using three different modes of locomotion - teleport, walk, and an automatic hover mode.
